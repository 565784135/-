二、window直接安装失败或启动失败



所以，老老实实用U盘安装吧

1、使用U盘安装
2、下载  https://www.deepin.org/download/ 


3、解压出  deepin-boot-maker.exe （镜像内是缩写名称 ）


然后放到同级目录，双击666，开玩笑，双击DEEPIN_B.EXE
选择镜像文件 deepin-15.6-amd64.iso
选择U盘，开始制作，（U盘剩余空间4G以上）
       



三、到此， 制作完成，已经完成了一半的进度啦，下面开始敲黑板啦！！！

1、分盘！！！

我的电脑 -> 右键 -> 管理 -> 磁盘管理 -> C盘或D盘 -> 右键 -> 压缩卷 ->  
推荐输入 40930（也就是40G）-> 右键压缩出来的未分配磁盘-> 新建简单卷 ->  
一直下一步搞定


注：
a、 当前系统为Windows 8或以上版本，请先关闭快速启动，否则无法安装。
https://jingyan.baidu.com/article/0320e2c19fe55c1b87507b30.html

b、Win10压缩卷提示磁盘空间不足，意思就是连续空白磁盘不足
https://blog.csdn.net/shentanyue/article/details/79452961

我把上面连接上的操作了一遍，清理回收站，还是不足，最后

Advanced SystemCare -> 工具箱 -> 磁盘清理 ，然后好了。我也不知为啥。

四、插入U盘，重启，进入启动菜单，选择U盘启动

我的是华硕，win10，按F2 进入的，下图有点不靠谱，自己试试吧





选择简体中文  -> 用户名密码 -> 时区：上海 -> 选中刚才分出来的40G的盘 ->开始安装
然后出去散散步抽根烟，5分钟就好了
固态盘就是快。
